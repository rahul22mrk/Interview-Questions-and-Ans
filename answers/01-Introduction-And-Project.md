# Q6. Tell me about a critical production bug you handled and how you resolved it.

## Interview Answer

One production issue I handled was related to OTP generation.

Some users were requesting OTPs repeatedly, which increased server load.

To solve this, I implemented Redis-based rate limiting with an expiry time, ensuring that users could request OTPs only after a defined interval.

This reduced unnecessary requests, improved server performance, and prevented abuse of the OTP service.
